РЕЖИМЫ:
:			Режим Команд.
i,I,a,A		Режим Вставки.
/,?,*,noh	Поиск.
v,V			Визуальный режим.
ESC			Выход из режимов в режим Навигации.
Ctrl c		Аналогично ESC
Ctrl [		Аналогично ESC

РЕЖИМ ВСТАВКИ:
i			Вставить текст до курсора.
I			Вставить текст с начала строки.
a			Добавить текст после курсора.
A			Добавить текст с конца строки.
o			Открыть новую строку ниже.
O			Открыть новую строку выше.

R			перейти в режим ввода с заменой текста (аналог insert). Символы под курсором заменяются на
			вводимые. При удалении вводимых символов возвращаются те которые были до ввода команды. Такой
			режим сохраняется до конца строки. При вводе новой строки (по нажатию Enter), происходит не
			переход на другую строку с тем же режимом замены текста, а создание новой строки.
r			заменить один символ. Заменяет символ находящийся под курсором на символ который следует за
			командой. При этом не происходит выхода из командного режима (не надо нажимать ESC после
			изменения текста). Например, команда ry - символ под курсором меняется на "y"
i			перейти в режим ввода с текущей позиции
I			переместиться в начало строки и перейти в режим ввода
a			перейти в режим ввода после курсора
А			переместиться в конец строки и перейти в режим ввода
o			перейти в режим ввода с новой строки под курсором
O			перейти в режим ввода с новой строки над курсором
x			удалить символ под курсором (<число>x удаляет указанное число символов начиная с того который
			находится под курсором)
X			удалить символ влево (удалить символ перед курсором)
d			используется совместно с командами перемещения. Удаляет символы с текущего положения курсора до
			положения после ввода команды перемещения. Пример:
dw			удаляет символы с текущего до конца слова. включая пробел после слова, чтобы удалить символы до
			конца слова, но оставить пробел используется команда de,
dE			удаляет символы с текущего до конца слова, включая символы пунктуации, но оставляет пробел после
			слова,
diw			удаляет слово под курсором,
dd			удалить текущую строку (вырезать)
d<число>d или <число>dd	стирание числа строк начиная с текущей,
db			удаляет символы с текущего до начала слова (удаление в обратном направлении)
d0			удаление символов с начала строки до текущего положения курсора
d$ или D	удаление символов с текущего положения курсора до конца строки.
с			команда аналогичная d, но после удаление переходит в режим ввода. (замена текста) Очень удобная
			альтернатива команде R
сс			команда удаляет текущую строку и переходит в режим ввода.
C			удаляет текст с текущего положения курсора до конца строки, аналогична команде с$
			(где $ - символ конца строки)
s			заменяет указанное количество символов (удалет указанное число символов и переходит в режим
			ввода). В отличии от команды "с" которая удаляет самое меньшее слово (cw), командой s можно
			удалить любое число символов. Например 4s удалит четыре символа начиная с того который находится
			под курсором.
S			удаляет всю текущую строку и переходит в режим ввода. Число перед командой показывает сколько
			нужно удалить строк начиная с текущей. Например 4S удалит четрые строки включая текущую.
yy			копирование текущей строки в неименованный буфер
y<число>y	копирование числа строк начиная с текущей в неименованный буфер
p			вставка содержимого неименнованного буфера под курсором. Поскольку в vim девять удаления. Можно
			вставить не только последнее удаление но и удаления сделанные ранее. Например "4p вставит под
			курсор содержимое четвертого удаления начиная с последнего. Также чтобы поменять местами два
			символа можно использовать комбинацию команд "удалить" -x (удаление в буфер) и "вставить"
			- p (вставить из буфера). Таким образом, поставив курсор на первую букву из двух которые
			необходимо поменять местами и нажав комбинацию клавиш xp мы совершим необходимые действия.
P			вставка содержимого неименнованного буфера перед курсором
J			слияние текущей строки со следующей
u			отмена последней команды
U			отмена всех последних изменений в строке
<Ctrl-r> или :redo		 вперед по изменениям
.			повтор последней команды
~			тильда меняет символы под курсором на противоположный регистр (заглавные в прописные и прописные
			в заглавные) может применяться числовой показатель. например команда пять символов начиная с
			текущего под курсором на противоположный регистр.

РАЗДЕЛИТЕЛИ:
:sp			Команда разделения экрана по горизонтали.
:sp [имя файла]		Команда открытия файла в горизонтальном экране.
:vsp		Команда разделения экрана по вертикали.
:vsp [имя файла]	Команда открытия файла в вертикальном экране.
ctrl+w+k	Вверх к разделенному окну.
ctrl+w+j	Вниз к разделенному окну.
ctrl+w+h	Влево к разделенному окну.
ctrl+w+l	Вправо к разделенному окну.
Сtrl-w o	развернуть окно
Ctrl-w c	закрыть
Ctrl-w s	разделить окно по горизонтали
Ctrl-w v	тоже, только по вертикали
Ctrl-w ]	разделить и перейти на определение чего-то, что под курсором.
Ctrl-w f	разделить и в новом окне открыть файл путь к которому находится под курсором.
Ctrl-w |	Сдвинуть в край вертикальное разделение окна
Ctrl-w _	Сдвинуть вниз горизонтальное разделение окна
Ctrl-w =	Нормализовать сдвинутые по вертикали или горизонтали разделения.
Ctrl-w +	Изменять размер между сплитами
Ctrl-w -	Изменять размер между сплитами
:wa 		Сохранить все открытые файлы.
:xa			Сохранить все измененные файлы и выйти.
:qa			Выйти из всех открытых файлов.
:qa!		Выйти без сохранения из всех открытых файлов.
:q			Выход из программы vim.
:w			Сохранить изменения в файле.
:wq			Команда сохранения изменений и выхода.
:x			Сохранить и выйти, если изменений не было в тексте, то выйти тогда.
:w [имя файла]		Сохраниться и создать файл.
:e [имя файла]		Открытие или создание нового файла в обычном режиме.
:e .			Вход в режим Netrw (дерево файлов и каталогов).
vim .			Запуск vim  в режиме Netrw.

НАСТРОЙКИ:
Данные настройки можно вводить в программе. А если необходимо, чтоб они сохранились, записываем их  в vimrc:

"Появятся номера строк.
set number	

"Показывает положение экрана относительно всего текста в %.
set ruler	

"Показывает строку состояния всегда.
set laststatus=2	

"Количество пробелов, которыми символ табуляции отображается при добавлении.
set tabstop=4	

"По умолчанию используется для регулирование ширины отступов в пробелах.
set shiftwidth=4	

"Копирует отступы с текущей строки при добавлении новой + автоматически выставляет отступы в «нужных» местах.
set smartindent		

Проверка орфографии на русском языке.
setlocal spell spelllang=ru

"Проверка орфографии на английском языке.
setlocal spell spelllang=en_us

"Отключение орфографии.
setlocal nospell

"Проверка орфографии через клавишу F4.
function! ToggleSpell()
if !exists("b:spell")
setlocal spell spelllang=ru,en
let b:spell = 1
else
setlocal nospell
unlet b:spell
endif
endfunction
nmap <F4> :call ToggleSpell()<CR>
imap <F4> <Esc>:call ToggleSpell()<CR>a

"Активизируем складки
set foldmethod=indent

"Задаём открытие/закрытие складок пробелом
nnoremap <space> za

"Возле курсора линия.
set cursorline

"Netrw древовидное отображение каталогов.
let g:netrw_liststyle = 3

"Перенос строки после 110 символов.
set textwidth=110

"Подсвечивание границы в 110 символов
set colorcolumn=110
highlight ColorColumn ctermbg=darkgrey

"Сохранить все открытые изменённые файл и Компиллировать проект с помощью клавиши F7 в режиме навигации и вставки
nnoremap <F7> :wa<cr>:make<cr>
inoremap <F7> <ESC>:wa<cr>:make<cr>a

"Запуск проекта через Alt+F7 в режиме навигации и вставки
noremap <a-F7> :!./dcdetali<cr>
imap <a-F7> <ESC>:!./dcdetali<cr>a

"Собераем make файл с помощью cmake в режиме навигации и вставки клавишами Alt+F9
noremap <a-F9> :!cmake -DCMAKE_BUILD_TYPE=Debug ..<cr>
imap <a-F9> <ESC>:!cmake -DCMAKE_BUILD_TYPE=Debug ..<cr>a

"Перемещаемся на вкладку влево с помощью F10
noremap <F10> gT
imap <F10> <ESC>gT

"Перемещаемся на вкладку вправо с помощью F12
noremap <F12> gt
imap <F12> <ESC>gt

"Поиск дополнительный файлов настроек .vimrc в которой запущен vim
set exrc

"Показать список функций в файле через клавишу F8
nmap <F8> ::TlistToggle<CR>

"Поддержка русской раскладки клавиатуры в режиме ввода команд.
set langmap=ёйцукенгшщзхъфывапролджэячсмитьбюЙЦУКЕHГШЩЗХЪФЫВАПРОЛДЖЭЯЧСМИТЬБЮ;`qwertyuiop[]asdfghjkl\\;'zxcvbnm\\,.QWERTYUIOP{}ASDFGHJKL:\\«ZXCVBNM<>
